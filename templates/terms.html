Service: AI Racing Engineer - F1 Car Builder

Purpose: This application provides an interactive F1 car building experience powered by AI-driven NPCs. You'll work with Dax, an AI racing engineer, to configure your Formula 1 car through natural conversation and receive real-time recommendations.

1. About This Service

AI Racing Engineer is an interactive web application that combines Formula 1 car customization with AI-powered natural language processing. Built with FastAPI, PostgreSQL, and advanced AI models (Groq API with Llama 3.1), this platform provides an engaging experience for racing enthusiasts and gamers.

Technology Stack

  • Backend: FastAPI (Python) with asynchronous processing
  • Database: PostgreSQL (Neon.tech) with AES-256 encryption at rest
  • AI Models: Groq API (Llama 3.1) for natural language processing and sentiment analysis
  • Authentication: OAuth 2.0 (Google, GitHub) + bcrypt password hashing
  • Security: TLS 1.3 encryption in transit, JWT tokens, httpOnly cookies
